### 📌 기록을 남기는 이유

- 기록을 차곡차곡 쌓아나가는 것은 성장의 원동력이 됩니다. 기록을 남기지 않으면 스스로 얼마만큼 해왔는지 체감하기 어렵습니다. TIL을 게임의 경험치와 레벨처럼 생각하면 배움이 더욱 즐겁게 느껴집니다. 🏃‍♂️
